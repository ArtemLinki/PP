**Датчики:**

**Питание:** USB от зарядника или блока питания 5V.

### Пример 3 — Сравнение товаров

---

**Пользователь:** Чем отличается ESP32-S3 от Raspberry Pi Pico W? Что лучше взять для проекта носимого устройства с датчиком пульса?

---

**ИИ-инженер:**
*[вызывает `compareProducts(["ESP32-S3-DEV", "RPI-PICO-W"])`]*

| Параметр | ESP32-S3 | Raspberry Pi Pico W |
|----------|----------|---------------------|
| Процессор | Xtensa LX7 × 2 @ 240 МГц | Cortex-M0+ × 2 @ 133 МГц |
| RAM | 512 KB + 8 MB PSRAM | 264 KB |
| Wi-Fi | ✓ 802.11n | ✓ 802.11n |
| BLE | ✓ 5.0 | ✓ 5.2 |
| USB нативный | ✓ (OTG) | ✓ |
| Цена | 1 290 ₽ | 890 ₽ |

**Вывод для носимого устройства:** Pico W — лучший выбор. Меньше потребляет энергию (важно для батарейки), C SDK и MicroPython отлично поддерживают I2C для MAX30102. ESP32-S3 мощнее, но избыточен для пульсоксиметра.

### Пример 6 — Техническая помощь с совместимостью

---

**Пользователь:** Могу ли я подключить модуль WS2812B ленту (5V) к ESP32 напрямую? Нужен ли резистор?

---

**ИИ-инженер:**

Да, подключить можно, но есть важный нюанс:

1. **Питание 5V:** ESP32 работает от 3.3V, но GPIO выдаёт 3.3V логику. WS2812B официально требует сигнал ≥ 3.5V — на практике большинство лент работает, но **не гарантированно**. Решение: использовать логический преобразователь 3.3V→5V или питать GPIO через буфер (например, 74HCT125).

2. **Резистор:** Да, **330–500 Ом в серию** на линию данных защитит от высокочастотных помех по кабелю. Это рекомендует Adafruit.

3. **Конденсатор:** 1000 мкФ / 6.3V между + и − питания ленты погасит всплески тока при старте.

4. **Ток:** 60 LED × 60 мА (max) = 3.6А. Питать ленту нужно от отдельного источника 5V/5A, **не от пина 5V Arduino/ESP32**.

Если хотите упрощённую схему без уровней — рекомендую взять ESP32-S3, он надёжнее тянет WS2812B на 3.3V благодаря более сильным GPIO-драйверам.

### Пример 7 — Рекомендация альтернатив при дефиците

---

**Пользователь:** Нет ли аналога HC-SR04, который точнее измеряет дистанцию?

---

**ИИ-инженер:**
*[вызывает `searchProducts("датчик расстояния точный")`, `compareProducts(["SEN-HCSR04", "SEN-VL53L0X"])`]*

Есть отличная альтернатива — **VL53L0X (ToF лазерный дальномер)**:

| Параметр | HC-SR04 | VL53L0X |
|----------|---------|---------|
| Технология | Ультразвук | Лазер ToF |
| Дальность | 2…400 см | 1…200 см |
| Точность | ±3 мм | ±3% |
| Угол | 15° | 25° |
| Зависит от цвета? | Нет | Нет |
| Интерфейс | Digital Trig/Echo | I2C |
| Цена | 180 ₽ | 520 ₽ |

**Вывод:** VL53L0X точнее, не зависит от температуры воздуха и работает через I2C (освобождает GPIO). Минус — дороже и дальность чуть меньше. Для роботов и дронов — однозначно лучше. Добавить в корзину?

## Часть 3 — Теги DIY-проектов в каталоге

Каждый компонент в каталоге помечен тегами вида `diy-<проект>`. Это позволяет ИИ и поиску точно находить компоненты по названию проекта.

| Тег | Проект |
|-----|--------|
| `diy-weather` | Wi-Fi метеостанция |
| `diy-robot` | 4WD-робот |
| `diy-pulse-oximeter` | Носимый пульсоксиметр |
| `diy-gps-tracker` | GPS-трекер с GSM |
| `diy-air-quality` | Монитор качества воздуха |
| `diy-gimbal` | 2-осевой стабилизатор |
| `diy-badge` | E-Ink умный бейдж |
| `diy-security` | Охранная система |
| `diy-irrigation` | Умный автополив |
| `diy-lora-sensor` | LoRa-сеть мониторинга |
| `diy-smart-home` | Умный дом (общее) |
| `diy-lighting` | Адресная подсветка WS2812B |
| `diy-clock` | Часы реального времени |
| `diy-monitoring` | Мониторинг энергопотребления |

### Как пользоваться в чате с ИИ:

Напишите в поле чата:
- _«Собери мне список компонентов для diy-robot»_
- _«Найди всё для метеостанции»_
- _«Что нужно для охранной системы с GSM?»_
- _«Подбери набор для начинающего до 2000 рублей»_
- _«Сравни ESP32-S3 и Pico W для носимого устройства»_

ИИ-инженер найдёт все нужные товары в каталоге, сравнит их и добавит в корзину по вашей команде.
