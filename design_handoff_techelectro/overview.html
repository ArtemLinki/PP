Mobile-first Next.js 14 + TypeScript, Mantine v7 для UI, Zustand для состояния, Axios + TanStack Query для данных. Интерфейсы сервисов и переключаемые Mock/Real реализации, DTO для всех сущностей.

01 · STACK

Выбранный стек

UI

Mantine v7 + Tabler Icons

Тёмная тема из коробки, готовая палитра под teal-акцент из Figma. Notifications, формы, лейаут.

STATE

Zustand + persist

Минимум бойлерплейта. Persist для корзины и auth-токена. Селекторы для derived state.

DATA

Axios + TanStack Query

HttpClient с интерцепторами, типизированный unwrap, обработка 401. Поверх — кеш React Query.

03 · SERVICES

Mock ⇄ Real через один интерфейс

UI знает только об IProductsService, ICartService и т.п. Какая реализация подставлена — решает createServices() по флагу из env.

services/types.ts

Контракт сервиса

export interface IProductsService { list(query?: ProductListQuery): Promise<Paginated<ProductDto>>; getById(id: ID): Promise<ProductDto>; getBySlug(slug: string): Promise<ProductDto>; } // два класса, один контракт class ProductsApiService implements IProductsService { … http.get(…) } class ProductsMockService implements IProductsService { … локальные данные }
services/index.ts

Фабрика, единственное место где принимается решение

export function createServices(overrides = {}): ServiceContainer { const useMocks = apiConfig.useMocks; const base = useMocks ? { products: new ProductsMockService(), … } : { products: new ProductsApiService(), … }; return { …base, …overrides }; // удобно подменять в тестах } export const services = createServices();
api/http-client.ts

Один axios-инстанс, авторизация, обработка 401

export class HttpClient { private instance: AxiosInstance; constructor(baseURL = apiConfig.baseUrl) { this.instance = axios.create({ baseURL, timeout: 15_000 }); this.instance.interceptors.request.use(cfg => this.attachAuth(cfg)); this.instance.interceptors.response.use(r => r, e => this.handleError(e)); } setAuthTokenProvider(fn: () => string | null) { … } setOnUnauthorized(fn: () => void) { … } get<T>(url, cfg?) { return this.instance.get(url, cfg).then(this.unwrap); } post<T, B>(url, body) { … } patch, delete … }
store/useCartStore.ts

Zustand + persist, мутации через сервис

export const useCartStore = create<CartState>()( persist( (set) => ({ cart: null, loading: false, error: null, refresh: async () => { set({ loading: true }); const cart = await services.cart.getCurrent(); set({ cart, loading: false }); }, add: async (p) => set({ cart: await services.cart.addItem(p) }), // update, remove, clear … }), { name: "techelectro.cart", storage: createJSONStorage(() => localStorage) }, ), );
